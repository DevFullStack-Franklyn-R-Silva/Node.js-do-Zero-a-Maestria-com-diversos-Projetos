# Utilizando o else

```txt
- O else é um complemento do if;
- Utilizamos no Handlebars para a exibição de outra parte do layout, caso a condição seja falsa;
- Isso nos dá mais flexibilidade ainda!
- A sintaxe é: {{#if alguma coisa}} ... {{else}} ... {{#/if}}
```
