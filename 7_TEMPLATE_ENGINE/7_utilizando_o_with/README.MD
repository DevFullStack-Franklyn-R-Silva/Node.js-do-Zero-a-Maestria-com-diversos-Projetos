# Utilizando o with

```txt
- O with nos permite abstrair um objeto;
- Ou seja, podemos acessar as propriedades sem nos referenciarmos
sempre ao objeto antes;
- A sintaxe é: {{#with objeto}} ... {{/with}}
- Desta maneira nossa código fica ainda mais simples;
```
